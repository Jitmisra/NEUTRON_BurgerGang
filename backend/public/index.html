HealthPrevent API

Welcome to the HealthPrevent API. This is the backend service for the HealthPrevent preventive healthcare platform.

API Endpoints

Authentication

POST /api/auth/register

Register a new user

POST /api/auth/login

Login to get an access token

GET /api/auth/me

Get current user profile

Health Metrics

GET /api/metrics

Get all user metrics

POST /api/metrics

Create a new health metric

Symptoms

GET /api/symptoms

Get all symptoms logged by the user

POST /api/symptoms

Log a new symptom

Health Journal

GET /api/journal

Get all journal entries

POST /api/journal

Create a new journal entry

Health Goals

GET /api/goals

Get all health goals

POST /api/goals

Create a new health goal

For detailed API documentation, please refer to the API documentation.
